Use cases
*********

Stocks
======

- See all his stocks
- See all rates
- Creating new stocks

Transactions
============

- See transactions history
- Create inner transaction
- Create outer transaction
- Exchange between his own stocks
- Transfer to other user's stock
- Revoke transfer transaction
